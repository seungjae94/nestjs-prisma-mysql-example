### 강의와 다르게 진행한 부분

1. 그룹 프로젝트에서 DB로 `MySQL`을 사용하기로 결정했기 때문에 `PostgreSQL`이 아닌 `MySQL` 을 사용했다.

2. 그룹 프로젝트에서 ORM으로 `prisma`를 사용하기로 결정했기 때문에 `typeorm`이 아닌 `prisma`를 사용했다.
